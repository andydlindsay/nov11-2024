### Primitive Types
* no properties (like objects or arrays have)
* stored directly in memory (RAM)
* cannot be changed once they are set

```js
const str = 'hello';
const num = 42; // 4.5
const bool = true;
const undefined = undefined;
const null = null;

const sym = Symbol();
const bigInt = 42n;
```
